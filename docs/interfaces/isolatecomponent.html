withContext

withContext: <ContextType>(type: Context<ContextType>, value: ContextType) => IsolateComponent
  • Defined in index.ts:143

Set context for isolated components.

param

The context type. This is the return value from React.createContext()

param

The value of the context, to set for testing.

Returns a new isolateComponent function that will include the specifed context, making it available to components that use useContext.

example
set a context value for testing: 
const isolatedWithContext = isolateComponent.withContext(MyContext, 'my context value')(<MyComponentThatUsesContext />)
